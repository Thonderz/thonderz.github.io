Temel HTML Kodu

HTML (HyperText Markup Language) web sayfalarını yapılandırmak için kullanılan işaretleme dilidir. Bu bölüm, HTML'in temel yapı taşlarını ve kullanımlarını açıklar.

  • <!DOCTYPE html>: HTML5 belgesi olduğunu belirtir.
  • <html lang="tr">: Sayfanın dilini Türkçe olarak belirler.
  • <head>: Sayfanın başlık ve meta bilgilerini içerir.
  • <body>: Sayfanın ana içeriğini barındırır.

Örnek HTML Kodu

<!DOCTYPE html><br> <html lang="tr"><br> <head><br> <meta charset="UTF-8"><br> <meta name="viewport" content="width=device-width, initial-scale=1.0"><br> <title>Basit HTML Sayfası</title><br> </head><br> <body><br> <h1>Merhaba Dünya!</h1><br> <p>Bu, basit bir HTML sayfasıdır.</p><br> </body><br> </html>

Kod Çıktısı

Merhaba Dünya! Bu, basit bir HTML sayfasıdır.
