Ders İçeriği

Bu ders, HTML (HyperText Markup Language) dilinin temellerini öğretir. HTML, web sayfalarını oluşturmak için kullanılan temel bir işaretleme dilidir.

Öğreneceğiniz konular:

  • HTML'in Temel Yapısı
  • HTML Etiketleri ve Kullanım Alanları
  • Sayfa Yapısı ve İçerik Düzenleme

Örnek HTML Kodu

<!DOCTYPE html><br> <html lang="tr"><br> <head><br> <meta charset="UTF-8"><br> <meta name="viewport" content="width=device-width, initial-scale=1.0"><br> <title>Basit HTML Sayfası</title><br> </head><br> <body><br> <h1>Merhaba Dünya!</h1><br> <p>Bu, basit bir HTML sayfasıdır.</p><br> </body><br> </html>
