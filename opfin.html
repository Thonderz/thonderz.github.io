HTML Temelleri

HTML (HyperText Markup Language), web sayfalarını oluşturmak için kullanılan bir işaretleme dilidir. Bu ders, HTML'in temel yapı taşlarını ve nasıl kullanılacağını öğretir.

Örnek HTML Kodu

<!DOCTYPE html>
<html lang="tr">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Basit HTML Sayfası</title>
</head>
<body>
<h1>Merhaba Dünya!</h1>
<p>Bu, basit bir HTML sayfasıdır.</p>
</body>
</html>

Açıklama

  • <!DOCTYPE html>: HTML5 belgesi olduğunu belirtir.
  • <html lang="tr">: Sayfanın dilini Türkçe olarak belirler.
  • <head>: Sayfanın başlık ve meta bilgilerini içerir.
  • <body>: Sayfanın içeriğini barındırır.
